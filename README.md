  <h1>Digikala Account tools</h1>
  <p>
    A simple gui windows program for extract account data and analyze the history of your purchases on Digikala.
  </p>
  </div>
  
## Requirements:
- .NET Framework 4.7.2 and greater

## NuGet packages
- [LiveCharts](https://github.com/Live-Charts/Live-Charts)
- [HandyControl](https://github.com/ghost1372/HandyControl)
